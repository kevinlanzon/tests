Junior Frontend Developer Test

We would like you to create a website that contains a login form with the following elements:

● Input element named “username”

● Input element named “password”

● Button element named “submit”

Then make the login form work, checking if the given username and password are correct against the following API:

https://api.myjson.com/bins/3t5s4

If the username and password pair are wrong, an error message will be displayed within the form. If they are correct, the form itself must be replaced by a loading spinner.

Instructions:

To build the project we require you to use HTML5, CSS3, Javascript and Responsive Web Design techniques. You are free to use any frameworks or libraries you please. We want to see your creative skills, you have the freedom to make this look as you please!

This test should take around 30 to 60 minutes.
